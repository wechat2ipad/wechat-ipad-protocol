# 使用建议

为了保证项目稳定运行，建议：

- 使用稳定服务器
- 使用稳定代理
- 保持账号正常使用习惯
- 不要高频异常调用接口
- 不要短时间批量操作账号
- 业务系统做好限流和重试
- 消息回调建议入库后异步处理
- AI 回复建议增加人工接管机制
- 多账号场景建议做好账号隔离
